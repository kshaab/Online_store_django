## Использование
Клонируйте репозиторий: 
```bash
git clone https://github.com/kshaab/Online_store_django
cd online_store_django
```
Установите зависимости и активируйте виртуальное окружение: 
poetry install
poetry shell

Запустите сервер разработки: 
python manage.py runserver


## Структура проекта
